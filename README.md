# 📦 Real-Time Inventory Dashboard

A real-time inventory analysis dashboard built using Streamlit.

---

## 🚀 Features

- 📊 Live data simulation
- 📦 Inventory tracking
- 🚨 Low stock alert system
- 📈 Category-wise analytics
- 🔄 Real-time updates

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy

---

## ▶️ How to Run

1. Install dependencies:
```bash
pip install -r requirements.txt

2. Run the app:
streamlit run app.py
